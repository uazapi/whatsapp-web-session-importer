Session Migration Connector
Extensão para backends autorizados

Importação de sessão com validação, chunks e limpeza local

1

Abra pelo painel

Use link direto ou bridge para abrir o WhatsApp Web com os dados da instância.

2

Capture a sessão

A extensão lê a sessão autenticada no WhatsApp Web sem expor dados internos ao SaaS.

3

Envie para a API

O payload é dividido em chunks, com hash SHA-256 por parte e validação no backend.

4

Finalize com segurança

Após a importação, a sessão local do navegador é limpa para evitar uso duplicado.

Projetada para suporte e painéis SaaSA integração usa link direto com client/token ou um fluxo com importKey e gateway do cliente.
